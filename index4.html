The New Standard of Bootable USB

เลิก Format แฟลชไดรฟ์
ซ้ำแล้วซ้ำเล่า

Ventoy คือเครื่องมือโอเพนซอร์สที่ให้คุณสร้าง Bootable USB เพียงครั้งเดียว จากนั้นแค่ "ลากไฟล์ ISO ไปวาง" ก็บูตได้ทันที จะใส่กี่ระบบปฏิบัติการก็ได้ในตัวเดียว

วิธีติดตั้ง Ventoy ทำไมต้องใช้?

ความต่างที่สัมผัสได้

วิธีเดิม (Rufus/Etcher)
  • ต้อง Format USB ทุกครั้งที่จะเปลี่ยน OS
  • 1 USB = 1 OS เท่านั้น
  • เสียเวลาแตกไฟล์ ISO นาน
วิธี Ventoy
  • ติดตั้ง Ventoy ลง USB แค่ครั้งเดียว จบ!
  • ใส่หลายไฟล์ ISO ได้ (Windows 10, 11, Ubuntu รวมกันได้)
  • แค่ Copy ไฟล์ ISO ไปวาง เหมือนเก็บไฟล์งาน
  • พื้นที่เหลือใช้เก็บข้อมูลอื่นได้ปกติ

เริ่มต้นใช้งานใน 3 ขั้นตอน

ง่ายจนคุณจะสงสัยว่าทำไมเมื่อก่อนต้องทำอะไรยุ่งยาก

01

ดาวน์โหลด & ติดตั้ง

โหลด Ventoy จากเว็บทางการ เสียบ USB แล้วกดปุ่ม "Install" โปรแกรมจะแบ่งพาร์ติชัน USB ให้คุณอัตโนมัติ

> Formatting USB...
> Installing Ventoy to MBR...
> Done.
02

ลากไฟล์ ISO ลงไป

เปิด My Computer จะเห็นไดรฟ์ชื่อ "Ventoy" ให้ Copy ไฟล์ .iso ของ Windows, Linux หรือ Tools ต่างๆ ลงไปวางดื้อๆ เลย

ISO
Windows11.iso → USB Drive (F:)
03

บูตแล้วเลือกเลย!

เสียบ USB เข้าคอม รีสตาร์ท กดปุ่ม Boot Menu จะเจอหน้ารายการของ Ventoy ให้เลือกเลยว่าจะบูตเข้าไฟล์ ISO ไหน

ข้อควรระวัง & ทริค

  • Secure Boot: คอมพิวเตอร์บางเครื่องอาจต้องปิด Secure Boot ใน BIOS ก่อนถึงจะบูต Ventoy ได้ (หรือเลือกเปิด Option Secure Boot ใน Ventoy ตอนติดตั้ง)
  • ชื่อไฟล์: อย่าตั้งชื่อไฟล์ ISO ที่มีเว้นวรรคหรือตัวอักษรแปลกๆ เพื่อความชัวร์ (เช่น `win 10.iso` → `win10.iso`)
  • อัปเดตง่าย: ถ้า Ventoy มีเวอร์ชันใหม่ แค่เสียบ USB เปิดโปรแกรมบนคอม กด Update ได้เลย ข้อมูลข้างในไม่หาย!
