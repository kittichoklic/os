การติดตั้ง Ventoy ลง USB

สำหรับ Windows (โหมด GUI)

01

1 ดาวน์โหลดและแตกไฟล์

ดาวน์โหลดแพ็คเกจติดตั้ง (เช่น ventoy-x.x.xx-windows.zip) จากเว็บไซต์ทางการ แล้วทำการแตกไฟล์ (Decompress/Unzip)

หมายเหตุ: โปรแกรม Ventoy2Disk.exe เป็นแบบ x86_32 รองรับทั้ง Windows 32-bit และ 64-bit (Intel/AMD)
02

2 รันโปรแกรมและติดตั้ง

เปิดไฟล์ Ventoy2Disk.exe เลือกอุปกรณ์ USB ของคุณ แล้วกดปุ่ม Install (ติดตั้งใหม่) หรือ Update (อัปเดตเวอร์ชัน)

MBR vs GPT: ตัวเลือก Partition style จะถูกใช้เฉพาะตอนกด Install เท่านั้น หากเป็นการกด Update ระบบจะข้ามขั้นตอนนี้ไป (รักษาข้อมูลเดิมไว้)

ข้อมูลเพิ่มเติมและข้อควรระวัง

สำหรับผู้ใช้ ARM หรือ X64 เฉพาะทาง (เวอร์ชัน 1.0.58+)

หากคุณต้องการใช้เวอร์ชันเฉพาะทาง Ventoy มีไฟล์แยกให้ในโฟลเดอร์ altexe ได้แก่:

  • Ventoy2Disk_X64.exe
  • Ventoy2Disk_ARM.exe
  • Ventoy2Disk_ARM64.exe

⚠️ วิธีใช้: ต้องคัดลอกไฟล์เหล่านี้ออกมาวางไว้ที่โฟลเดอร์ชั้นนอก (ที่เดียวกับ Ventoy2Disk.exe ตัวหลัก) ก่อนใช้งาน

การติดตั้งลง Hard Disk (ไม่ใช่ USB)

เพื่อความปลอดภัย โดยปกติ Ventoy จะแสดงเฉพาะ USB Drive เท่านั้น หากต้องการติดตั้งลง Hard Disk ในเครื่อง:

  1. ไปที่เมนู Option
  2. เลือก Show all devices

!!! ระวัง: โปรดตรวจสอบให้แน่ใจว่าเลือกไดรฟ์ถูกต้อง การเลือกผิดจะทำให้ข้อมูลใน Hard Disk หายทั้งหมด

แก้ปัญหาการติดตั้งล้มเหลว

หาก Ventoy2Disk.exe ทำงานล้มเหลวตลอด คุณสามารถใช้วิธี Ventoy LiveCD แทนได้ (ดูรายละเอียดในคู่มือขั้นสูง)

การ Format หลังติดตั้ง

หลังจากติดตั้งเสร็จแล้ว พาร์ติชันที่ใช้เก็บข้อมูล (พาร์ติชันแรก) สามารถ Format ใหม่ได้ด้วยตัวเอง รองรับระบบไฟล์หลากหลาย: exFAT / FAT32 / NTFS / UDF / XFS / Ext2 / Ext3 / Ext4

แปลและเรียบเรียงจากเอกสาร Official ของ Ventoy
